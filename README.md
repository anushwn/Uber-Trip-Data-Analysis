# 🚖 Uber Trip Data Analysis

This project explores Uber trip data using Python to uncover key revenue and operational insights. The analysis includes data cleaning, feature engineering (trip duration, fare per km, revenue per minute), and KPI generation such as total revenue, average fare, peak hours, and payment type contribution.

The findings highlight clear peak-demand hours, strong correlation between trip distance and fare, and variations in revenue per minute that suggest pricing and allocation opportunities. This project demonstrates practical skills in Pandas, data visualization, business insight generation, and analytical thinking relevant to real-world data analyst roles.
